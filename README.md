# Modern Web Starter Project

A sleek and visually appealing starting point for vanilla web applications carefully following premium, dynamic design aesthetics.

## Project Structure

- `index.html` - The HTML backbone containing semantic elements, valid structure, and SEO best practices including titles, meta tags. Links to Google Fonts to fetch the Inter typeface.
- `styles.css` - Custom styles employing CSS variables across a modern dark theme, animated presentation, and rich visual appeal via glassmorphism elements.
- `script.js` - Foundation for DOM-focused interactivity.

## How to use

Simply open the `index.html` file in your favorite modern browser, or spin up a local development server like `npx serve` or using the VS Code Live Server extension to start hacking on your project immediately!
